1. Data Collected

  • Server and Channel IDs
  • Alert-related message content and embeds

We do not collect usernames, messages from DMs, or personal information.

2. How We Use Data

Data is used only to detect alerts and send role-based notifications. It is processed temporarily in memory and discarded unless needed for configuration.

3. Data Sharing

We do not share or sell any data to third parties.

4. Data Retention

Temporary data is discarded immediately. Configuration data (e.g., role or channel IDs) is stored only for functionality and can be deleted upon request.

5. Your Rights

You may remove the bot from your server at any time. You may also request deletion of stored configuration data by contacting us via GitHub or Discord.
